# Le Camの不等式

## ステートメント

$(\mathcal{X}, \mathcal{A})$ を可測空間として，$P, Q$ をその上の確率測度とする．$P$ と $Q$ の全変動距離を $d_{\mathrm{TV}}(P, Q)$, Hellinger距離を $H(P, Q)$ と表す．このとき

$$
\frac{1}{2} H(P, Q) \leq d_{\mathrm{TV}}(P, Q) \leq H(P, Q)\sqrt{1 - \frac{H^2(P, Q)}{4}}
$$

が成り立つ．
